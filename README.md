# OpenCode Mobile
<img width="1206" height="2622" alt="Simulator Screenshot - iPhone 16 Pro - 2026-03-26 at 11 29 26" src="https://github.com/user-attachments/assets/b81e6d3c-dd10-48ba-903a-d4a01067c954" />

A React Native mobile app for OpenCode — an AI coding assistant.

## Features

- Connect to a local OpenCode server
- View and manage coding sessions
- Real-time session status monitoring
- Dark/light mode support

## Tech Stack

- **Framework**: Expo (React Native)
- **UI**: HeroUI Native with Uniwind (Tailwind v4)
- **Navigation**: React Navigation with expo-router
- **AI Integration**: @opencode-ai/sdk

## Setup

```bash
npm install
npx expo start
```

## Run on Device

```bash
# iOS
npm run ios

# Android
npm run android
```

## Requirements

- Node.js 18+
- Expo SDK 54
